# Presentation Workspace

This directory contains the slide deck, speaker notes, and generated assets for the AXIOM-Mobile research presentation.

## Contents

| File | Description |
|------|-------------|
| `SLIDE_DECK_v1.md` | Full slide deck in Markdown format (Marp/reveal.js compatible) |
| `SPEAKER_NOTES.md` | Per-slide speaker notes with timing guidance and Q&A prep |
| `assets/generated/` | Charts and figures generated by the build script |

## Slide Format

The deck is written in Markdown with `---` slide separators, compatible with:

- **Marp** -- `marp SLIDE_DECK_v1.md --pdf`
- **reveal.js** -- drop into a reveal.js project as `index.md`
- **Keynote / PowerPoint** -- convert via Marp (`--pptx`) or Pandoc

## Generating Assets

Presentation figures (learning curves, latency comparisons, scorecard) are produced by:

```bash
python ml/scripts/build_presentation_assets.py
```

Output lands in `assets/generated/`. The slide deck references these files by relative path.
